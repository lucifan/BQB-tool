根目录执行命令 node app.js 即可运行

============================备注=====================================

1. RGB即颜色的RGB三个数字
2. 对齐指的是输入文字的对齐方式，左对齐为l，右对齐为r
3. 一行为一个素材
4. 预览图，生成图分为根目录下两个文件夹"生成图"和"预览图"
5. 图片命名按照配置文件的顺序依次为"1.jpg","2.jpg"...
6. 配置文件的命名为"file.in"注意用utf-8编码
7. 注意要用英文的逗号
8. 由于代码中没有接口url所以无法正常运行嘿嘿XD

=========================配置信息格式================================

素材配置信息的格式为：

表情名,文字大小,R,G,B,字体,对齐,x坐标,y坐标


一张图片的配置信息示例如下：

表情名：除了恋爱没什么好谈的
文字大小：24px
颜色：(4,4,4)
字体：jianti
对齐：右对齐
x坐标：180
y坐标：237
则对应的配置信息应该为：
除了恋爱没什么好谈的,24,4,4,4,jianti,r,180,237
